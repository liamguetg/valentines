# How to Pull a Baddie This Valentine's Day

Welcome to the ultimate guide to charm your valentines. This isn't just a project—it's your ticket to Valentine's Day success. You're welcome ;)\
Made using simple HTML/CSS, JavaScript, Anime.js, Tailwind CSS


## Project

Project is essentailly a valentines day card, with moving photobooth style pictures. The next page has a place to write something cute and a menu for valentines dinner.

## Customize It

- **Change the messages**: Update the `messages` array in `script.js` to match your love language
- **Change the pictures/ change the sound**: Works bests with exactly 24 pictures. Toss new images into the `images` folder and update `imagePaths` or add another mp3 file for button click sounds.
- **Tweak styles**: Go wild in `style.css` or tweak Tailwind classes in `index.html`
